# Teste Prático — Módulo 2

Repository containing the solution to a practical assessment from the second module of programming studies.

## Objective

The work applies the concepts introduced during the module in a small, self-contained implementation. The repository is kept public as evidence of learning progression and problem-solving practice.

## Review locally

1. Inspect the source files and identify the main entry point.
2. Use the compiler or runtime appropriate to the language in the repository.
3. Run the program from a terminal.
4. Compare the result with the requirements of the original exercise.

## Scope

This is an academic assessment, not a production application. The code reflects the knowledge and constraints of the module in which it was created.
